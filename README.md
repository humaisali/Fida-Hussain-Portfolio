## Overview

A clean, fully responsive personal portfolio website developed for a client as a freelance project under **SkyTech Developers**. The site showcases the client's professional background, skills, projects, and contact information in a modern single-page layout.

**Live:** [fida-hussain-portoflio.vercel.app](https://fida-hussain-portoflio.vercel.app/)

---

## Features

| Feature | Description |
|---|---|
| **Hero Section** | Animated introduction with name, title, and call-to-action |
| **About Section** | Professional summary and background |
| **Skills Section** | Visual display of technical skills and tools |
| **Projects Section** | Showcased work with descriptions and links |
| **Contact Section** | Contact form and social media links |
| **Fully Responsive** | Optimized for desktop, tablet, and mobile |
| **Fast Performance** | Built with Vite for near-instant load times |

## Getting Started

### Prerequisites

- Node.js v18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/humaisali/Fida-Hussain-Portfolio.git

# Navigate into the project
cd Fida-Hussain-Portfolio

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Production Build

```bash
npm run build
```
